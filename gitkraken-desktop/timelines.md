This page describes the legacy GitKraken Timelines integration in GitKraken Desktop and the remaining toolbar controls that were available before the product sunset. Use it only if you need to understand older workspace behavior or remove the Timelines button from the GitKraken Desktop toolbar in an environment that still shows it.

**Requirements and limits**
- GitKraken Timelines sunset at the end of 2022, so this page covers legacy behavior only.
- Use this page for historical reference or to remove the Timelines button from the toolbar in older environments.
- Timelines access depended on the toolbar button and a GitKraken account connection.
- Any Timelines controls described here may not exist in current GitKraken Desktop releases.

GitKraken Timelines are designed to show major milestones on a continuous line that represents time.

<figure>
  <img src='/wp-content/uploads/timelines/overview/timelines-overview@2x.png' class="help-center-img img-bordered" alt="Timeline milestones on a continuous line that represents time">
  <figcaption style="text-align: center; color: #888">Timeline milestones on a continuous line that represents time.</figcaption>
</figure>

Each milestone highlights what is important, when the milestone happens, and the primary goals of the milestone.

## How to access Timelines <a id="access-timelines"></a>

GitKraken Timelines is automatically connected to your GitKraken account. No additional configuration is needed. You can access Timelines with the `Timelines` button in the GitKraken toolbar.

<figure>
  <img src='/wp-content/uploads/timelines/integrations/gitkraken/gitkraken-integration@2x.png' class="help-center-img img-bordered" alt="Timelines button in the GitKraken toolbar">
  <figcaption style="text-align: center; color: #888">Timelines button in the GitKraken toolbar.</figcaption>
</figure>

You can also select a specific timeline to open from the drop-down menu. Clicking the circle <span class='sr-only'>circle </span><kbd><i class="far fa-circle"></i></i></kbd> icon next to a timeline will set it to the default for this repository. The current default will be shown as a filled <span class='sr-only'>green circle </span><kbd><i class="far fa-dot-circle"></i></kbd> icon.

<figure>
  <img src='/wp-content/uploads/timelines/integrations/gitkraken/default@2x.png' class="help-center-img img-bordered" alt="Default timeline option in the Timelines drop-down menu">
  <figcaption style="text-align: center; color: #888">Default timeline option in the Timelines drop-down menu.</figcaption>
</figure>

## How to remove Timelines from the toolbar

To remove Timelines from the toolbar, go to <em class='context-menu'>Preferences <i class='fa fa-caret-right'></i> UI Preferences</em> and clear _Show Timelines button in toolbar_.

[Learn more](/timelines/overview/) about Timelines.
